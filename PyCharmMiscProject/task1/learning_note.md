## Task0 学习内容
- 什么是多层感知机（MLP）？其结构是怎样的？  
    答：
    MLP是一种基础的前馈神经网络。它由输入层、若干隐藏层和输出层组成，数据从输入层进入，经过隐藏层逐层加工，最后由输出层给出预测结果。

    核心结构：输入层->隐藏层1->激活函数->隐藏层2->激活函数->输出层
    
    输入层负责接收原始数据；隐藏层负责对输入特征进行加权组合和抽象提取；激活函数通常放在隐藏层之后，用于引入非线性能力，如果没有激活函数，多层线性叠加后本质上仍然只是一个线性变换，模型难以学习复杂的分类边界；输出层负责给出最终预测结果，其样貌和答案相似。


- 数据在神经网络中扮演哪些角色？（数据集的 split 和处理）  
    答：
    数据在神经网络中作为训练集和检测集。在监督学习中，一条数据通常包括两部分，**特征**和**标签**。特征是输入给模型的信息，标签是正确答案。  
    为了判断模型是否真正学会，而不是只是记住训练数据，通常要把数据集进行划分(即**dataset split**)，常见划分为：  
    训练集 training set  
    验证集 validation set  
    测试集 test set  
      
    训练集用于训练模型。模型通过反复看这些数据，通过计算损失函数和反向传播来更新参数。
    验证集通过调整模型设置，例如learning rate,隐藏层数量，batch size，训练轮数 epochs等，帮助判断模型设置是否合适。
    测试集用于最终评估模型效果。测试集的数据不参与训练。


- 噪声是什么？特征是什么？标签是什么？
    噪声是数据中的干扰因素，通常由测量误差、随机扰动、数据质量问题引起，导致一些样本的位置发生偏移，导致两类数据混在一起，产生干扰。  

    特征是输入给模型的信息。

    标签是样本对应的正确答案。

- Batch size 是什么？为什么堆叠成 Batch 可以提高运算速度？  
答：Batch size指的是*每次训练时，一次喂给模型多少个样本*。利用Batch可以把多条数据堆叠成一个大矩阵的运算，而CPU/GPU很擅长一次性处理大矩阵，因此使用Batch可以减少python循环开销，更好利用硬件的并行能力，尤其GPU

- 神经元是什么？  
答：神经元是神经网络里的*小计算单元*。  
其流程图如下：

    接收一组输入  
    ↓  
给每个输入乘一个权重  
↓    
全部加起来  
↓   
再加一个偏置  
↓  
经过激活函数  
↓  
输出一个结果  

    y = activation(w1x1 + w2x2 + w3x3 + ... + b)

    其中  
x1, x2, x3 ... 是输入  
w1, w2, w3 ... 是权重  
b 是偏置  
activation 是激活函数  
y 是这个神经元的输出
